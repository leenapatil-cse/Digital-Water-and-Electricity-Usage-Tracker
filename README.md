 💧 Digital Water & Electricity Usage Tracker

📌 Description
The Digital Water & Electricity Usage Tracker is a data-driven web application designed to help users monitor and predict their household resource consumption.

Unlike traditional systems that provide delayed feedback through utility bills, this project enables proactive decision-making by forecasting future usage using time-series analysis.

🚀 Key Features
- Forecasts future water and electricity consumption
- Allows users to set daily consumption limits
- Generates alerts if predicted usage exceeds defined limits
- Interactive dashboard with visual insights
- Helps promote sustainable and responsible resource usage

🛠️ Tech Stack
- Python  
- Streamlit (for web interface)  
- Pandas (data handling)  
- Prophet (time-series forecasting)  
- Plotly (data visualization)

⚙️ How It Works
1. User inputs a future date and consumption limits  
2. Historical data is processed using Pandas  
3. Prophet model predicts future consumption  
4. System compares prediction with user limits  
5. Alerts are generated if limits are exceeded  
6. Results are displayed using interactive charts.
